# 📊 Sales Analysis Dashboard – Power BI

## 📌 Project Description

This project is a sales analysis dashboard built using Power BI.

The goal was to apply my data analysis skills to explore sales performance, simulate profitability, and extract basic business insights using DAX and data modeling.

The dashboard includes:

- Total Sales
- Simulated Total Profit
- Margin %
- Sales Trend over time
- Sales by Category
- Sales, Profit & Margin comparison
- Top 10 Products by Sales

---

## 🧠 Profit Simulation

The original dataset did not include profit data.

To make the analysis more complete, profit was simulated using fixed margins by category:

- Technology → 20%
- Furniture → 15%
- Office Supplies → 10%

This allowed me to calculate total profit and margin percentage.
